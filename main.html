Etymology

The notation commemorates two mathematicians whose work inspired its structure.

The name of this googologic notation honors two mathematicians: Wacław Sierpiński (Poland) and John H. Conway (United Kingdom), both of whom made deep contributions to mathematical logic, set theory, and large number notations. The notation combines the iterative spirit of Sierpiński’s transfinite constructions with Conway’s chained-arrow operations.

Definition

Core operator: the lambda prefix encoding Conway chained arrows.

To introduce this notation, we begin with the lambda operator. Placing the symbol λ before a natural number \( n \) defines an operation based on Conway’s arrow notation:

\[ \lambda n = n \rightarrow n \rightarrow n \rightarrow \cdots \rightarrow n \quad (\text{n times}) \]

That is, \( \lambda n \) represents the Conway chained-arrow operation applied n times to the number \( n \) itself.

Multilambda Operation

Stacking lambdas to obtain auto-iterative growth.

We define a higher-level operator by stacking lambdas:

\[ \lambda \lambda n = \lambda (\text{n times})\, n \]

This Multilambda Operation grows vastly faster than the simple λ-operation, representing self-iteration of λ applied n times.

Alphabetic Hierarchy

Generalize with Greek-letter indexed meta-iterations.

To generalize the notation, we introduce a recursive family of functions using Greek letters. The first level is defined as:

\[ \omega(n) = \lambda (\text{n times})\, n \]

Then we extend this with a secondary parameter to represent repeated iteration:

\[ \omega(n, m) = \omega^{(m)}(n) \]

where \( \omega^{(m)} \) denotes m iterations of ω applied to n. This process can be further generalized:

\[ \omega(n, m, a) = \left( \omega^{(m)}(n) \right)^{(a)} \]

so that each new variable introduces an additional meta-iterative layer.

Alphabetic Escalation

Advancing letters creates higher auto-iteration degrees.

By advancing through the Greek alphabet — from \( \omega \) to \( \alpha \), \( \beta \), \( \gamma \), and beyond — each successive letter introduces a higher self-iterative degree of the previous operation. Thus, SCLN defines a transfinite-like hierarchy of hyperoperations, where every new Greek letter represents a function whose growth rate transcends all those before it.

Formal Recursive Definition

A compact recursive template that anchors the notation.

\[ f_0(n) = n \]
\[ f_{k+1}(n) = f_k(n) \rightarrow^{(n)} f_k(n) \]

Here \( \rightarrow^{(n)} \) denotes n nested Conway arrows. From this we obtain:

\[ \boxed{ \lambda n = f_n(n), \quad \lambda \lambda n = f_n^{(n)}(n), \quad \omega(n, m) = f_n^{(m)}(n) } \]

For the general alphabetic hierarchy we may write:

\[ \Omega_{\alpha}(n_1, n_2, \ldots, n_k) = \Omega_{\alpha-1}^{(n_k)}(n_1, n_2, \ldots, n_{k-1}) \]

where \( \alpha \) represents a Greek index corresponding to the current hierarchical depth.
